## 安装

``` cmd
$ yarn add @fe6/water
```

## 使用

- 通过 npm 安装依赖，引入

``` js
import water from '@fe6/water';
import '@fe6/water/dist/water.css';
```

- CDN 外部引用

``` html
<script src="https://unpkg.com/vue"></script>
<script src="https://unpkg.com/@fe6/water"></script>
```

## 特性

- 提炼自企业级中后台产品的交互语言和视觉风格。
- 开箱即用的高质量 `Vue` 组件。
- 使用 `TypeScript` 构建，提供完整的类型定义文件。

## 支持环境

- 现代浏览器和 IE9 及以上。
- 支持服务端渲染。

## 开发的环境

- node.js >= 11.0.0
- npm >= 6.0.0
